# Chat em Tempo Real com Node.js e Socket.IO

Projeto de chat em tempo real usando **Node.js**, **Express** e **Socket.IO**, permitindo múltiplos usuários se conectarem e trocarem mensagens instantaneamente.

## Tecnologias
- **Node.js** – ambiente de execução JavaScript no servidor  
- **Express** – framework para criação de rotas HTTP  
- **Socket.IO** – comunicação em tempo real via WebSocket  
- **HTML/CSS/JavaScript** – interface e lógica do frontend
